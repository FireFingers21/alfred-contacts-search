# <img src='Workflow/icon.png' width='45' align='center' alt='icon'> Contacts Search

Search & view fields in Contacts cards with Alfred

## Setup

This workflow works with Alfred's [Contacts Viewer](https://www.alfredapp.com/help/features/contacts/#viewer). Enable it by checking `Open Contacts in Alfred` in the Features > Contacts > Contacts preferences.

## Usage

Search Contacts details via the `cs` keyword. Type to filter.

![Using the Search Contacts keyword](Workflow/images/about/keyword.png)

* <kbd>↩</kbd> View Contact card.
* <kbd>⌘</kbd><kbd>↩</kbd> Open Contact card in Contacts app.

Contacts are always searchable by Full Name and Nickname, while searching by Job Title, Organization, Phone Number, Email Address, Street Address, and Notes is configurable from the [Workflow’s Configuration](https://www.alfredapp.com/help/workflows/user-configuration/).

![Filtering Contacts results](Workflow/images/about/filter.png)

Configure the [Hotkey](https://www.alfredapp.com/help/workflows/triggers/hotkey/) for faster triggering. For more advanced access, the [External Trigger](https://www.alfredapp.com/help/workflows/triggers/external/) is also available.

Optionally override where contacts are opened with the `URL Handler` field in the [Workflow’s Configuration](https://www.alfredapp.com/help/workflows/user-configuration/). This can be another app or an [External Trigger](https://www.alfredapp.com/help/workflows/triggers/external/) from another workflow.
